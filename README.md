# ReversaLog - Sistema de Logística Reversa

Sistema web para gerenciamento de logística reversa com Firebase e exportação em Excel.

## Funcionalidades
* Retorno de Materiais
* Triagem de Caixas
* Saída de Fornecedores
* Exportação em Excel (últimos 30 dias)
* Integração Firebase em tempo real

## Como usar
1. Abra o arquivo `index.html` no navegador
2. Preencha os formulários
3. Clique em "Salvar" para sincronizar com Firebase
4. Clique em "Exportar Excel" para baixar os dados

## Tecnologias
* HTML5, CSS3, JavaScript
* Firebase Realtime Database
* XLSX (exportação Excel)
